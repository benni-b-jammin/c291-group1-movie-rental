# Movie Rental Application using SQL Server
CMPT 291 X01L-Team 1, W2026

## Overview
This project utilizes an SQL Server database to run a mock movie rental application using Windows forms. 

## Setup
Some important setup notes:
- our program utilizes the files `RentalDB_create.sql` and `RentalDB_insert.sql` that requires the user to make a database entitled `RentalDB` set up in SQL Server.

- For database access by the Windows Forms application, create the following credentials for `RentalDB` (used so all team members have the same access):
```
username: superuser
password: dbaccess
```
